# gittuf Documentation

gittuf's documentation is structured that each type of user (e.g. consumer,
contributor, and maintainer) needs to read their section, and the sections that
come before it. For example, a consumer need only read [Part 2: gittuf for
Consumers], but a maintainer should read the sections for all roles, e.g. [Part
2: gittuf for Consumers], [Part 3: gittuf for Contributors], and [Part 4: gittuf
for Maintainers].
