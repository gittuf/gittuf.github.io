# gittuf Documentation

gittuf's documentation is divided into five parts:

1. Preface
2. Installation
3. Using gittuf (on a day-to-day basis)
4. Administering gittuf
5. The gittuf GitHub App

If you are a contributor and do not need to manage gittuf policy, you need only
read parts one, two, and three.

If you are an administrator, we suggest that you read the documentation in order
and entirely, so that you can confidently write your own gittuf policy. However,
you may also read the sections that you deem interesting/necessary.

At a minimum, we suggest you read parts one through four.
